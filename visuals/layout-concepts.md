# Layout Concepts

## Investor Deck Map
Use a dark, high-contrast platform galaxy: Arsenal in the center, six core primitives around it, and revenue paths highlighted.

## Founder Map
Use a practical systems map: route -> page -> service -> edge function -> table -> revenue/risk.

## Sales PDF
Use the Business Spaces revenue engine as the hero: pain point -> plan -> ROI -> proposal -> agents/automations -> outputs -> portal/export -> subscription/setup fee.

## Internal Engineering Map
Use swimlanes for frontend, services, edge functions, database, admin, and external providers.

## Non-Coder Map
Use visual metaphors: operating system, command center, digital workforce, map room, marketplace, control plane, vault, factory.
