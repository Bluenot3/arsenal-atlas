# Route Map Visual Guide

Use `data/route-map.csv` to create route visuals.

## Recommended Route Clusters

- Public acquisition: `/`, SEO pages, public agents, public Arsenals, programs.
- Activation: `/dashboard`, `/agents`, `/z-engine`, `/files`, `/instruction-lab`.
- B2B revenue: `/business-spaces`, `/business-spaces/:id/*`.
- Distribution: `/marketplace`, `/discover`, `/community`, `/arsenals`, `/experiences`.
- Execution: `/automation`, `/flows`, `/schedules`, `/flex-os`, `/integrations`.
- Monetization: `/pricing`, `/partners/dashboard`, `/revenue`.
- Governance: `/admin`, `/admin/pages`, `/admin/partners`, `/labs`.

## Visual Treatment

- Use a subway map for user journeys.
- Use thicker lines for revenue-critical routes.
- Use dashed lines for redirects and legacy/labs routes.
- Use lock icons for authenticated/admin routes.
- Use coin markers for pricing, credits, paid Arsenals, Business Spaces, and referrals.
